## 题解：[20191203]#0050 Pow(x, n)
- **题干**   
实现pow(x, n)函数，计算x的n次方。    
示例：    
  ```
  // e.g.1
  Input: 2.00000, 10
  Output: 1024.00000
  // e.g.2
  Input: 2.10000, 3
  Output: 9.26100
  // e.g.3
  Input: 2.00000, -2
  Output: 0.25000
  Explanation: 2^(-2) = 1/2^2 = 1/4 = 0.25
  ```
  说明：
  - -100.0 < x < 100.0
  - n是一个32位有符号整数，大小范围为[-2^31, 2^31-1]
