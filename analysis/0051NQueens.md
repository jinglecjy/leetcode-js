![8皇后问题的一种解法](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2018/10/12/8-queens.png)   
给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。   
每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。      
示例：    
  ```
  // e.g.1
  Input: 4
  Output: [
  [".Q..",  // 解法 1
    "...Q",
    "Q...",
    "..Q."],

  ["..Q.",  // 解法 2
    "Q...",
    "...Q",
    ".Q.."]
  ]
  ```
